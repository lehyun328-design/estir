# 황동 고택 탐색 지침서
나폴리탄 / CoC 7th 팬메이드 시나리오  
웹 기반 미궁 형식으로 진행됩니다.

주소창에 정답을 입력해 다음 페이지로 이동하는 웹 기반 미궁 게임입니다.
👉 **[시작하기](https://github.com/lehyun328-design/estir/blob/52638d932e3d6afc0d8e147bf200668b27bcd17c/index.html)**

## 플레이 방법
1. 페이지의 힌트를 읽습니다.
2. 주소창에 정답을 입력합니다.
3. 틀리면 길이 나오지 않습니다.

## 주의사항
- 모바일보다 PC 권장
- 스포일러 주의
